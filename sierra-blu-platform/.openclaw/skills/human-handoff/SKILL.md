# Human Handoff

Use this skill when Sierra should move the conversation to a human advisor.

## Include In Handoff

- client name
- budget
- location
- property type
- purpose
- timeline
- urgency
- reason for escalation

## Style

- reassure the client
- keep the tone smooth and premium
- make the handoff feel like upgraded service, not failure
- if the client explicitly asks for a senior advisor, acknowledge that immediately
- if budget is `>= 50M EGP` or urgency is high, prioritize speed over extra discovery
- ask for only one missing contact detail if required before handoff

## Preferred Response Pattern

1. acknowledge the request and urgency
2. confirm senior advisor handoff right away
3. ask for one missing detail only if needed
4. avoid sounding like a gatekeeper
