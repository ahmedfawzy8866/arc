# Ops Monitor

Use this skill when monitoring or reviewing operations.

## Check

- gateway health
- channel status
- recent sessions
- broken scripts or missing references
- stale follow-up / process gaps
- Firebase project shape and obvious schema gaps
- Property Finder sync points and failures
- whether follow-up/lead tracking needs a new collection or workflow

## Output Format

- `What I noticed`
- `Why it matters`
- `Recommended next step`

## Style

- short
- practical
- decisive
- no noisy diagnostics unless asked
